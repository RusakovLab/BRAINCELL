rainCell Examples

This folder contains fully runnable demonstration models for BrainCell.

Each example is self-contained and can be launched by double-clicking Run.bat.

The purpose of these examples is to:

Verify correct installation

Demonstrate core BrainCell capabilities

Provide minimal working templates for new users

How to Run

Open the desired example folder.

Double-click Run.bat.

Wait for NEURON to start and the model to execute.

No manual setup inside NEURON is required.

If an error appears, verify:

Mechanisms are compiled (build_mechs.bat)

NEURON is correctly installed

You are running from the official BrainCell folder structure
